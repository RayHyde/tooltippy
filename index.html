Tooltippy

a flexible, CSS-only tooltip that can contain full HTML

Download from Github

Out on the net are some wonderful CSS-only tooltip solutions that convert attributes such as titles, names or data to the tooltip text. However, for a project I needed to include HTML in the tooltip.

As I soon found out, a centred tooltip is nice in design, but won't wash in real life, with right and left-aligned text that can sit close to the top or bottom of a page and that also needs to be responsive.

So I've made this tooltip flexible, changing its position by adding class names

.tooltippy-above +
.tooltippy-align-left This is the HTML content
Bold, Italic
.tooltippy-above This is the HTML content
Bold, Italic
.tooltippy-above +
.tooltippy-align-right This is the HTML content
Bold, Italic
.tooltippy-below +
.tooltippy-align-left This is the HTML content
Bold, Italic
.tooltippy-below This is the HTML content
Bold, Italic
.tooltippy-below +
.tooltippy-align-right This is the HTML content
Bold, Italic
